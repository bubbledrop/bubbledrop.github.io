MLLP

  • MLLP stands for Minimal Lower Level Protocol
  • Straightforward encapsulation: <SB>HL7 message<EB><CR>
  • The newer version defines some reliability measures:
    • The receiver acks every received message with the format <SB><ACK><EB><CR>.
    • A NAK is also possible in the same format.
    • A new message will not be sent until an ack is received for the previous message.
  • In ASCII: SB = 0x0b, EB = 0x1c ACK = 0x06, NAK = 0x15.
